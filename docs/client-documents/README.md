# 📁 Client Documents - Upload Guide
**⚠️ CONFIDENTIAL - These files are git-ignored and will NOT be committed to version control**

---

## Folder Structure

### requirement-files/
Upload the regulatory requirement PDFs here.
**Examples:**
- Cabinet Decision No. 74.pdf
- UAE AML/CFT Regulations.pdf
- UN Security Council Resolutions.pdf

### internal-process-docs/
Upload the bank's internal policy documents here.
**Examples:**
- UAE Implementation of TFS.docx
- Bank AML Policy v3.pdf
- Customer Due Diligence Procedure.xlsx

### sample-excel-sheets/
Upload the manually-worked Excel sheets provided by client.
**Examples:**
- sample output excel sheet.xlsx
- Cabinet Decision 74 - Manual Analysis.xlsx

### sample-output-templates/
Upload the desired output format templates.
**Examples:**
- Compliance Report Template.xlsx
- Dashboard Mockup.png

---

## ⚠️ Security Notice
- These files contain sensitive banking information
- They are listed in .gitignore and will NEVER be pushed to git
- Only store on the development machine or secure shared drive
- Delete from local machine after processing/testing
